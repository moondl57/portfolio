Event Reservation App

Event registration and managing app
during the COVID-19 pandemic

The Challenge

The Boston Chinese Evangelical Church(BCEC) has re-launched in-person services. In effort to follow the public health guidelines, they decided to only accept a limited number of people by registration.

My Role

User Research

  • User interviews
  • User flow
  • Site map

Design

  • UI

User Research

I interviewed BCEC's web developer who collected opinions from the members of BCEC. I was able to come up with the following pain points that needed to be addressed.

Pain point 1

The sign-up page has poor translation accessibility.

Pain point 2

4 registration links are used for 2 campuses and the events are not organized.

Pain point 3

There is no waitlist feature allowing people to join if someone else cancels their registration.

Screenshot of current website link to registration

Personas

I created 3 personas representing the needs of the project. These personas were referred to throughout the entire product development process. Personas are an educated representation of my intended target audience. Personas are not an actual individual.

Solution

  • Create a pre-registration website accessible and manageable by a single address.
  • Incorporate multi-language support on pages that members have access.
  • Include a waitlist feature to events that reached maximum capacity.

Site map

I designed the site map and user flow in a way that allows the user to easily navigate and complete their needs with the least number of essential steps.

User flow and wireframing

Created a user flow to specify the main features the user would go through. This was used to estimate diffoculties the user may experience throughout the use of the product.
The following digital sketches were made early during the design phase and went through modifications to improve usability.

UI Design

This project uses two fonts, Poppins and Noto Sans SC. Poppins is a geometric sans-serif typeface with great curves and round-corners which gives the website a modern look.
Noto Sans SC provides legible chinese characters that are sans-serif, following the Poppins feel, yet in a different language.

The color palette is inspired by BCEC's logo. Accompanying it with a soft white creates a gentle tone to the entire website.
